Unibot

Este sistema foi projetado para simplificar e otimizar o preenchimento de formulários online, especialmente para clientes que precisam automatizar suas tarefas repetitivas.

link do modelo para fins didáticos

Propósito do projeto

O intuito deste projeto é ajudar empresas e profissionais a economizar tempo e reduzir erros em tarefas repetitivas de preenchimento de formulários online. Ao automatizar esses processos, os usuários podem focar em atividades mais estratégicas e de maior valor, aumentando a eficiência e produtividade de suas operações. Se você está interessado em saber mais ou precisa de uma solução de automação personalizada, não hesite em entrar em contato comigo!

O Sistema oferece as seguintes funcionalidades:

Automação de Navegação Web: Utilizando a biblioteca Selenium, o sistema interage com navegadores web para preencher formulários de forma automática.
Processamento de Dados: Com pandas, o sistema lê e manipula dados de planilhas Excel, garantindo que todos os campos necessários sejam preenchidos corretamente.
Interface Intuitiva: A interface gráfica foi construída com customtkinter, proporcionando uma experiência de usuário amigável e fácil de usar. Com botões de "Iniciar", "Pausar" e "Finalizar", o controle da automação está na ponta dos dedos.
Monitoramento em Tempo Real: O sistema exibe o tempo decorrido, o número de formulários concluídos e os erros encontrados, permitindo um acompanhamento preciso do processo.
Tratamento de Erros: Erros são tratados de forma eficiente, garantindo que o processo continue com os próximos formulários, e os detalhes dos erros são exibidos para fácil diagnóstico.

Como Funciona:

Início da Automação: Carregue a planilha Excel e inicie a automação.
Login Automático: O sistema realiza login no site com as credenciais fornecidas.
Preenchimento de Formulários: Dados são extraídos da planilha e inseridos nos formulários automaticamente.
Monitoramento: A interface exibe o progresso em tempo real.
Finalização: Um relatório detalhado é gerado ao final do processo
