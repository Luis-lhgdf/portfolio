Project Overview

The FakePinterest project is a recreation of the Pinterest page, allowing users to add and share images. Developed in Python with Flask, it utilizes SQLAlchemy to connect to a SQLite database and Flask-Bcrypt to encrypt passwords.

The main routes include:

Homepage: Login process and redirection to create an account.
Create Account: Account creation with password and email verification.
Profile: View of the user's profile, with the option to view other profiles through the feed.
Logout: Possibility to log out of the account at any time.
Feed: Central page where images shared by users are displayed.
View: Allows viewing the profile that shared an image and adding the image to the user's own profile.
Add: Sharing of images on the user's profile.

For the page's visual, CSS and HTML were used. This project was developed based on the Python Impressionador course by Hashtag Treinamentos, with additional implementations made by me.
