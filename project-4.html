Python World

Este projeto é uma aplicação interativa desenvolvida em Python para praticar exercícios de Python 3 apresentados nos "Mundos" do curso Curso em Vídeo. O usuário podem escolher entre os diferentes "Mundos" do curso e responder aos exercícios correspondentes, com correções automáticas fornecidas pela API OpenAI.

Project Link

Propósito do projeto

Este projeto foi desenvolvido com o propósito acadêmico e nostálgico de relembrar os exercícios dos "Mundos" 1, 2 e 3 do curso de Python do Curso em Vídeo com Gustavo Guanabara. O projeto oferece uma interface gráfica interativa que permite aos usuários escolher um mundo e responder aos exercícios correspondentes, recebendo correções automáticas por meio da integração com a API OpenAI

O código oferece as seguintes funcionalidades:

Seleção de Mundos: Os usuários podem escolher entre os "Mundos" 1, 2 e 3 do curso clicando nos respectivos botões na interface gráfica.
Visualização de Exercícios: Após escolher um mundo, os exercícios correspondentes a esse mundo são exibidos na interface.
Resposta e Correção Automáticas: Os usuários podem responder aos exercícios por meio de uma caixa de texto na interface. As respostas são então enviadas para a API OpenAI para correção automática.
