Python World

This system is an interactive application developed in Python to practice Python 3 exercises presented in the "Worlds" of the Curso em Vídeo course. Users can choose between the different "Worlds" of the course and respond to the corresponding exercises, with automatic corrections provided by the OpenAI API.

Project Link

Project Overview

This project was developed with the academic and nostalgic purpose of recalling the exercises from "Worlds" 1, 2, and 3 of the Python course by Curso em Vídeo with Gustavo Guanabara. The project offers an interactive graphical interface that allows users to choose a world and respond to the corresponding exercises, receiving automatic corrections through integration with the OpenAI API

The code offers the following functionalities::

World Selection: Users can choose between "Worlds" 1, 2, and 3 of the course by clicking on the respective buttons in the graphical interface. Exercise Visualization: After choosing a world, the exercises corresponding to that world are displayed in the interface. Automatic Response and Correction: Users can respond to exercises through a text box in the interface. The responses are then sent to the OpenAI API for automatic correction.
